All in one Loyalty Rewards and redeem Program for Website and POS Odoo Apps

All in one Loyalty and Rewards program for Webshop and Point of sales odoo apps is used to give Sale,POS and website loyalty redemption points for every purchase to your customers from there screen.Customer can also redeem this loyalty points for other purchase from there screen. Every single purchase from the Sale, POS and Website records the Loyalty Rewards based on configuration setup on there backend and those rewards will be redeemed on Sale,POS and Website order from screen order easily. Reward redeems visible on applied Sale, POS and Website order so it will be helpful for see history of the reward point redemption.

Features

Loyalty Point Configuration

Apply loyalty point based on loyalty configuration rule.

Earns loyalty points

Customer can earns loyalty points with his/her every purchase in Sale,POS and Website.

Define Loyalty Points

Define Loyalty based on product category(Sale, POS, Website) and purchase amount.

Redeem loyalty points

Customer can redeem loyalty points as a discount for other purchase order in Sale, POS and website.

History of Loyalty Points

User can see there loyalty points transactions history.

Validations for Sale, POS and Website

User have get some validation messages when they give wrong input or enter more than loyalty points also avoid selection of customer.
