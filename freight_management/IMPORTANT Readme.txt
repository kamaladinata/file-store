Please note that the Air freight operation is handeled in a separate module.
You will find the module folder under the directory of this module, the Air freight module named: "kware_freight_air", please follow the following steps after you download the module files from the download link:

1. Download the module files normally from the download link as guided during your purchase
2. Extract the compressed module files
3. Under the Main module folder, you will find a folder named "kware_freight_air"
4. Please cut this module and paste it to your main Odoo addons folder
5. Copy the main freight management module folder (named: "freight_management") to your Odoo main addons directory
6. Procceed to installing the two modules one by one
