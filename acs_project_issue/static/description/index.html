Track Issues/Bugs Management for Projects

This application allows you to manage the issues you might face in a project like bugs in a system, client complaints or material breakdowns. It allows the manager to quickly check the issues, assign them and decide on their status quickly as they evolve.

Odoo Remove Project Issue module in v11 so we migrated same functionality from v10 to v11 in this module.

Project Issue Timesheet

When we work on any Issue we need to enter timesheet details also in odoo.

But many time because of work load we forget to enter timesheet entry and that can cause loss in revenue. To avoid such mistakes and loss we have implemented simple functionality in this module. When any user start work on any Issue they will have to click on start and when work is completed or stopped they have to click stop button thats it. It will automatically create timesheet entry.
