All in One Disable Followers

Generally in the odoo automatically partners(customers, vendors, contacts) added as the followers. so our module restricts that.This module disables the partners automatically added as followers. This module includes,

1) Confirmation Quotation: When you confirm the quotation the partners not added as the followers.

2) Validate Invoice/Bill/Credit Note/Debit Note: When you validate the invoice/bill/credit note/debit note the partners not added as the followers.

3) Send By Email: When you press send by email button in the sale order/quotation, purchase order/request for quotation & invoice/credit note the partners not added as the followers.

4) Create/Update Picking: When you create or update the picking the partners not added as the followers.

Features

You can restrict to add the partners(customers, vendors, contacts) as the followers.

Restrict the customer's add as the followers in the confirmation of the quotation.

Restrict the partner's add as the followers in the create/update of the picking.

Disable the customer's add as the followers in the validation of the invoice/credit note.

Disable the vendor's add as the followers in the validation of the bill/debit note.

Restrict the partners add as the followers in the send by email button in the sale order/quotation, purchase order/request for quotation & invoice/credit note.

No special configuration required, install it, use it.

It reduces human efforts.

In the follower's list partners add as the followers in the inventory.

In the follower's list partners not add as the followers.

v 13.0.1

Confirmation Quotation Workflow

Go to settings => general settings to restrict the partners automatically added as followers.

In the quotation, you can see the followers list.

Now, confirm the quotation.

The sale order is created.

In the follower's list customers add as the followers in the sale order.

In the settings => general settings tick the tickbox "Disable to add followers by confirm Quotation" to restrict the partners automatically added as followers.

In the quotation, you can see the followers list.

Now, confirm the quotation.

The sale order is created.

In the follower's list customers not add as the followers in the sale order.

Validate Invoice/Bill/Credit Note/Debit Note Workflow

In the invoice, you can see the followers list.

Now, validate the invoice.

In the follower's list customers add as the followers in the invoice.

In the settings => general settings tick the tickbox "Disable to add followers by validate Invoice/Bill" to restrict the partners automatically added as followers.

In the invoice, you can see the followers list.

Now, validate the invoice.

Now, the invoice in the "Open" state.

In the follower's list partners not add as the followers in the invoice.

Now, validate the credit note, you can see the followers list.

In the follower's list partners not add as the followers in the credit note.

Now, validate the bill, you can see the followers list.

In the follower's list partners not add as the followers in the bill.

Now, validate the vendor credit note, you can see the followers list.

In the follower's list partners not add as the followers.

Send By Email Workflow

In the quotation/sale order, we press "Send By Email".

Now we send the email, you can see the recipient's/customer's name.

In the follower's list customers add as the followers.

In the settings => general settings tick the tickbox "Disable to add followers by send by email" to restrict the partners automatically added as followers.

In the quotation/sale order, you can see the followers list.

We press "Send By Email".

Now we send the email, you can see the recipient's/customer's name.

In the follower's list customers not add as the followers.

In the request for quotation/purchase order, you can see the followers list.

We press "Send By Email".

Now we send the email, you can see the recipient's/partner's name.

In the follower's list partners not add as the followers.

In the invoice, we press "Send & Print".

Now we send the email, you can see the recipient's/partner's name.

In the follower's list partners not add as the followers.

In the credit note, we press "Send & Print", you can see the followers list.

Now we send the email, you can see the recipient's/partner's name.

In the follower's list partners not add as the followers.
