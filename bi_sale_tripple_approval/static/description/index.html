Sale Order Tripple Approval Process

This app helps you to tripple approval process for sale order while confirming sale order

Key Features:

  • Sale Order approval process: Sale Manager -> Finance Manager -> Director
  • After Install,This App will add approval buttons on sale order based on configuration in company's sale tripple approval setting.Enable Three Step Approval . Let's say SO Double validation amount = 1000 and SO is more then 1000 then it require approval for assign user , then if Finanace validation amount = 5000 then if SO amount is more then 5000 it will require two level approval process for assign user and then if Director validation amount =10000 then if SO amount is more than 10000 it will require three level approval process for assign user.
  • You can set three step verification approval process on Sale Orders
  • You can set a limit amount on Sale Order as Two and Three Step Verification
  • If Order's Total Amount exceeds the pre-defined finance limit,then Flow will be Two Step purchase validation process.
  • If Order's Total Amount exceeds the pre-defined director limit of three step,then Flow will be Three Step validation process.
  • Email Notification will sent to Next Level Person to Approve
