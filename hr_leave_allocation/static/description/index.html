Employee Bulk Leave Allocation

This application is used to allocate leaves to multiple employees

Basic Features
  • Using this app, Hr Manager can allocate leave to an employee in bulk
  • Leaves allocated to the employee in 'Approve' or 'To Approve' state
  • Hr Manager can also retry to allocated the if leaves are not allocated to all employee.
  • Hr Manager 'Allocate' or 'Cancel' only those allocations which are in 'draft' state
